

# ⚖️🤖 JurisAI – AI-Powered Legal Assistant

🚀 **JurisAI** is a multilingual AI-powered legal chatbot designed to make legal information **accessible, interactive, and easy to understand**.
It combines **AI, NLP, and modern web technologies** to deliver smart legal assistance in real-time.

---

## ✨ Key Features

🔹 **Multilingual Support**
Automatically detects user language and responds accordingly

🔹 **Voice Interaction**
Ask legal questions using speech recognition

🔹 **Audio Responses**
Listen to answers with text-to-speech functionality

🔹 **Intelligent Legal Retrieval**
Powered by FAISS vector store and HuggingFace embeddings for accurate context

🔹 **Persistent Chat History**
Maintains conversation history for better user experience

---

Legal information is often **complex and difficult to access**.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Sreoshi170/JurisAI-AI-Powered-Legal-Assistant.git
cd JurisAI-AI-Powered-Legal-Assistant
```

---

### 2️⃣ Create Virtual Environment

```bash
python -m venv myvenv
```

Activate:

```bash
myvenv\Scripts\activate
```

---

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

### 4️⃣ Setup Environment Variables

Create a `.env` file:

```env
GROQ_API_KEY=your_api_key_here
```

⚠️ Never share your API keys publicly

---

### 5️⃣ Run the Application

```bash
streamlit run app.py
```
